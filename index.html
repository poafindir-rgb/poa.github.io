Проект 1 — «Актирование работ ЖКХ (до/после)»

Веб-приложение для УК и подрядчиков: фиксирует выполненные работы, проверяет качество с помощью ИИ и формирует отчётную документацию (акт и иллюстрированный отчёт).

  • Реестр заявок с фильтрами по статусам (Черновик / Нужны фото / На проверке / Принято) и поиском.
  • Создание заявки с AI-классификацией типа работ и чек-листом фото-доказательств.
  • Фото «До/После» через drag-and-drop + наглядное сравнение и контроль ракурсов.
  • AI-анализ: проверка комплектности, авто-описание изменений, недостающие доказательства.
  • Генерация PDF: иллюстрированный отчёт и акт выполненных работ.
Мини-архитектура
  • Frontend: заявка + загрузка фото “до/после”.
  • AI: чек-лист, анализ доказательств, текст отчёта/акта.
  • Экспорт: PDF (акт + отчёт).

Проект 2 — «Нейро-диспетчер УК / Facility Desk»

B2B приложение для УК: принимает обращения жильцов, уточняет данные по чек-листу, классифицирует заявки, назначает SLA и ведёт цикл “обращение → задача → выполнение → аналитика”.

  • AI-сервис: классификация, чек-лист, уточнения, инструкции мастерам.
  • Роли: Resident / Dispatcher / Technician / Manager.
  • SLA и приоритизация: матрица приоритетов, визуальные таймеры.
  • Аналитика: нагрузка бригад, причины аварий, соблюдение SLA.
Мини-архитектура
  • Frontend: роли + канбан + SLA.
  • AI: классификация + вопросы + подсказки.
  • Данные: демо-данные + сохранение в браузере.

Проект 3 — «ОСС-ДокМастер»

Инструмент для УК и ТСЖ: подготовка ОСС (очно/заочно/очно-заочно) — повестка, документы, проверки логики, реестры уведомлений, голосование, архивный пакет.

  • Конструктор ОСС: параметры дома, тип, повестка из шаблонов.
  • AI-генерация: уведомления, бюллетени, протоколы.
  • AI-compliance: кворум, даты, приложения, логические несостыковки.
  • Экспорт: PDF + ZIP “пакет для архива/проверок”.
Мини-архитектура
  • Конструктор: мастер 3 шага + библиотека повесток.
  • AI: генерация + проверка логики.
  • Экспорт: PDF/ZIP.
